## Key

### Editor

+ ```t``` : 1x1 칸 입력
+ ```r``` : 3x3 칸 입력
+ ```Enter``` : Change Block
+ ```Space``` : Move Cursor Far
+ ```g``` : Test Map

### Game

+ ```Up``` : Jump
+ ```Space``` : Dash
+ ```Enter``` : Skill
+ ```Esc``` : Menu
